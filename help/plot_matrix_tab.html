Plot Matrix

User can Plot the raw data matrix generated from the previous tab, this is a simple PCA plot.

  • The "Options" tab gives the user the ability to choose the feature for coloring the points and which two Principle Components to plot.
  • The "Plot" tab has the interactive PCA plot and the user is able to:
    • Zoom-in
    • Zoom-out
    • Select a certain region of the plot
    • One click on one of the features on the right can remove the points belonging to that feature
    • Double click on one of the features on the right can remove all the other points and keep only the ones belonging to that feature.
  • The "Download Plot" tab appears only after plotting, and in that tab, the user can download a static PDF version of that plot, or an interactive HTML version.
